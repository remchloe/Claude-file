# Claude-file

由 Claude 生成的文件和聊天记录存储仓库。

## 目录结构

```
├── chat-records/          # 聊天记录
│   ├── summaries/         # 对话摘要
│   └── full-logs/         # 完整对话
├── generated-files/       # 生成的文件
│   ├── scripts/           # 脚本代码
│   ├── documents/         # 文档
│   └── configs/           # 配置文件
├── assets/                # 其他资源
└── README.md
```

## 使用说明

| 目录 | 用途 |
|------|------|
| `chat-records/summaries` | 保存与 Claude 对话的关键摘要 |
| `chat-records/full-logs` | 保存完整对话记录 |
| `generated-files/scripts` | 存放生成的脚本代码 |
| `generated-files/documents` | 存放生成的文档/报告 |
| `generated-files/configs` | 存放生成的配置文件 |
| `assets/` | 存放图片等资源文件 |
